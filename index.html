Blog Post on text-wrap: balance

0. Introduction

This blog explores a CSS property, text-wrap: balance, that adjusts line-wrapping styles.

What is it?

text-wrap is a CSS property that determines how to break lines
text-wrap: balance wraps text to balance the length of lines.

Why use it?

This property improves visual consistency of text blocks and can be useful to enhance readability.

Limitations

The algorithm is computationally expensive so it would only take effect for text under six lines.

1. What does text-wrap: balance do?

Setting the text-wrap property to "balance" aims to balance the empty space at the end of each line. It attempts to make the lines as close to the same length as possible. This prevents situations where lines in a text block have very different length.
